# BB Learn — proposal

A single-page proposal for **BB Learn**, BrainBack's intelligence layer for ed-tech: a voice-native teaching layer that speaks like a mentor, renders concepts on screen as it talks, and corrects student work in real time.

`index.html` is a self-contained page (fonts and the two production-session clips are embedded inline). Published via GitHub Pages.

Built on the BrainBack design system — strict monochrome, Geist Sans/Mono, pixel grid.
